전문 기능 일람

⚡

실시간 WebSocket

FastAPI 비동기 엔진으로 구동되는 초저지연 양방향 통신. 게임 상태가 즉시 모든 플레이어에게 동기화됩니다.

🏠

룸 매니지먼트

최대 4인 방 생성 및 입장, 호스트 자동 승계, 빈 방 자동 삭제까지 완벽한 룸 라이프사이클 관리.

💬

인게임 채팅

방별 실시간 채팅과 최근 20개 히스토리 제공. 입장 즉시 이전 대화를 확인할 수 있습니다.

🎯

플레이어 동기화

위치, 점수, 생존 상태를 실시간으로 브로드캐스트. 모든 클라이언트가 항상 최신 상태를 유지합니다.

🔌

Ping / Pong Keepalive

60초 타임아웃과 자동 핑퐁 heartbeat으로 유령 연결 없이 안정적인 세션을 보장합니다.

📡

REST API 로비

방 목록 조회, 생성, 삭제를 위한 REST 엔드포인트. /docs에서 Swagger UI로 바로 테스트 가능.

WebSocket 연결 예시

client.js — NEXUS ARENA
// 1. 서버에 연결
const ws = new WebSocket('ws://localhost:8000/ws/플레이어닉네임');

// 2. 게임 룸 입장
ws.send(JSON.stringify({
  "type": "join_room",
  "room_id": "abc123"
}));

// 3. 플레이어 이동 전송
ws.send(JSON.stringify({
  "type": "player_move",
  "position": { "x": 120, "y": 340 }
}));

// 4. 실시간 이벤트 수신
ws.onmessage = (e) => {
  const data = JSON.parse(e.data);
  console.log('이벤트:', data.type);
};
